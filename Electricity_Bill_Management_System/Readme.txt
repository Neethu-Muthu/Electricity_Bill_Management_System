HOW TO RUN THE PROJECT

After Starting Apache and MySQL in XAMPP, follow the following steps.

1st Step: Extract the file

2nd Step: Copy the main project folder(ebill)

3rd Step: Paste in xampp/htdocs/

4th Step: Open a browser and go to URL “http://localhost/phpmyadmin/”

5th Step: Then, click on the databases tab

6th Step: Create a database naming “ebillsystem” and then click on the import tab

7th Step: Click on browse file and select “ebillsystem.sql” file which is inside the folder DATABASE FILE

8th Step: Click on go.

After Creating Database,

9th Step: Open a browser and go to URL “http://localhost/ebill/index.php ”

For admin panel, go to URL: http://localhost/ebill/login.php
